# Лежандр — MindVector Mobile Client

Мобильный клиент для системы отслеживания прогресса в решении задач MindVector.

## Особенности

- 🧮 **Решение задач** — отслеживание времени, озарения, вопросы, подсказки
- 📚 **Библиотека** — источники и задачи с тегами и пагинацией
- 🏷️ **Модерация** — пользовательские источники и теги с модерацией
- 📊 **Статистика** — XP, стрик, активность
- 🤖 **AI Помощь** — подсказки от разных AI-персон (Кот Базис, Петрович, Лежандр)
- 📸 **Камера** — фотографирование условий и решений с OCR
- ✨ **Мотивация** — вдохновляющие тексты в зависимости от контекста
- 💰 **Финансы** — баланс, пополнение через ЮKassa, история транзакций

## Технологии

- **Flutter 3.x** — UI Framework
- **Riverpod** — State Management
- **GoRouter** — Navigation
- **Dio** — HTTP Client
- **Flutter Secure Storage** — Token Storage

## Модерация контента

Пользователи могут создавать собственные источники и теги, которые проходят модерацию:

- **pending** — на модерации (видно только автору)
- **approved** — одобрено (видно всем)
- **rejected** — отклонено

Владелец может редактировать свои pending-источники и теги.

## Мотивация

Модуль мотивации показывает вдохновляющие тексты в разных контекстах:
- Утро/день/вечер — разное время суток
- Начало сессии — мотивация на решение
- Достижения — поздравления
- Стрик — поддержка цепочки дней (1-90+ дней)

## Подготовка к релизу в RuStore

### Необходимые ресурсы

1. **Иконки приложения:**
   - `mipmap-mdpi/ic_launcher.png` (48×48)
   - `mipmap-hdpi/ic_launcher.png` (72×72)
   - `mipmap-xhdpi/ic_launcher.png` (96×96)
   - `mipmap-xxhdpi/ic_launcher.png` (144×144)
   - `mipmap-xxxhdpi/ic_launcher.png` (192×192)

2. **Скриншоты для магазина:**
   - Минимум 2 скриншота
   - Рекомендуемый размер: 1080×1920 или 1920×1080
   - Формат: PNG или JPEG

3. **Тексты:**
   - Краткое описание (до 80 символов)
   - Полное описание (до 4000 символов)
   - Ключевые слова для поиска
