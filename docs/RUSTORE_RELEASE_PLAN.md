## 4. Процесс публикации

### Шаг 1: Подготовка APK
```bash
# Настроить подпись в android/app/build.gradle.kts
# Создать keystore
keytool -genkey -v -keystore lezhandr-release.jks -keyalg RSA -keysize 2048 -validity 10000 -alias lezhandr

# Собрать release APK
flutter build apk --release
```

### Шаг 2: Регистрация в RuStore
1. Создать аккаунт разработчика на https://developer.rustore.ru
2. Заполнить профиль разработчика
3. Добавить новое приложение

### Шаг 3: Загрузка ресурсов
1. Загрузить иконки
2. Загрузить скриншоты
3. Заполнить описания
4. Указать категорию и рейтинг

### Шаг 4: Загрузка APK
1. Загрузить `app-release.apk`
2. Заполнить версию и изменения
3. Отправить на модерацию

### Шаг 5: Модерация
- Обычно 1-3 рабочих дня
- Возможны запросы на доработку
